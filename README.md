## Реализация

### Загрузка и предобработка данных
- Загружены train (74 659 объектов) и test (49 352 объектов) выборки из JSON
- Извлечены признаки из колонки `features`: удалены скобки, кавычки, пробелы
- Найдено 1 546 уникальных значений признаков
- Выбраны топ-20 самых частых (Elevator, CatsAllowed, HardwoodFloors и т.д.)
- Созданы 20 бинарных признаков + 2 числовых (bathrooms, bedrooms) = **22 признака**
- Применена очистка данных: отфильтрованы выбросы (1-й и 99-й перцентиль по цене), bedrooms < 5, bathrooms < 5

### Собственная реализация линейной регрессии (`customLinReg`)
Реализована с несколькими методами оптимизации:
- **Аналитический**: `w = (XᵀX + λI)⁻¹Xᵀy` (с использованием псевдообратной матрицы)
- **SGD**: Стохастический градиентный спуск с настраиваемым числом эпох
- **Batch GD**: Пакетный градиентный спуск
- **Mini-batch GD**: Мини-батчевый градиентный спуск с настраиваемым размером батча

Поддержка регуляризации:
- **Ridge (L2)**: штраф = λ × ||w||₂²
- **Lasso (L1)**: штраф = λ × ||w||₁ (недифференцируема в нуле, решается итеративно)
- **ElasticNet**: комбинация L1 и L2

Детерминированный режим через параметр `random_state`.

### Метрики и оценка
- Собственная функция `my_R2` для расчёта коэффициента R²
- Функция `get_model_table` для генерации таблиц с MAE, RMSE, R² для train/test
- Сравнение собственных реализаций со sklearn (близкие результаты)

### Нормализация признаков
Собственные реализации:
- **MinMaxScaler**: x_scaled = (x - x_min) / (x_max - x_min)
- **StandardScaler**: x_scaled = (x - μ) / σ

Проверена эквивалентность со sklearn через `np.allclose()`.
